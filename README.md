# CS4375

## Python Warm-up activity
* Task: Create a simple python Chatbot that would provide 5 responses to user inputs. 4 user inputs are known, and for any other user input a default response is given and the the chatbot terminated.


## Pre-shell exercise
* Task: Create a python script that will run 4 processes that execute 4 programs, waits for them to complete, and exits/terminates the script (along with the processes).


## Process Scheduling Simulation
* Task: Execute given python file (process-run.py), answer questions 1-5 from Ch.4 of book, and describe any noticable differences between python versions 2 and 3.
